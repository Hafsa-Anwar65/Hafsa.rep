# Age Group Prediction System

A simple Python program that predicts a user's age group based on the age entered.

## Features

* Takes age as input
* Classifies users as Child, Teenager, Adult, or Senior Citizen
* Uses Python dictionaries and `range()` for classification

## Technologies

* Python 3

## How to Run

```bash
python age_prediction.py
```

## Example

```text
Enter age: 22
Predicted Age Group: Adult
```

## Concepts Used

* User Input
* Dictionaries
* Loops
* Conditional Statements
* `range()` Function
